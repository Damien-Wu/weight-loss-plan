## 组件设计

### 导航栏组件

```
+------------------------------------------+
| Logo | 首页 | 饮食 | 运动 | 进度 | 常见问题 |
+------------------------------------------+
```

### 页脚组件

```
+------------------------------------------+
| 版权信息 | 免责声明 | 联系方式 | 返回顶部 ↑ |
+------------------------------------------+
```

### 计划卡片组件

```
+----------------------------------+
|           卡片标题               |
|                                  |
| [图标] 内容项目1                 |
| [图标] 内容项目2                 |
| [图标] 内容项目3                 |
|                                  |
|           查看详情 →             |
+----------------------------------+
```

### 日期选择器组件

```
+------------------------------------------+
| < 上一周 | 第X周 (X月X日-X月X日) | 下一周 > |
+------------------------------------------+
| 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
+------------------------------------------+
```

这些布局设计将作为HTML/CSS实现的基础，确保网站结构清晰、用户体验良好，并且在不同设备上都能正常显示和使用。
