# 30天减肥计划网站

这个仓库包含一个完整的30天减肥计划网站，专为26岁男性（身高175cm，体重85kg，目标减重10kg）设计，考虑到用户只能在晚上运动，中午需要在公司附近餐厅就餐的限制。

## 网站预览

![30天减肥计划网站预览](https://example.com/preview.png)

## 特点

- 基于科学原理的30天减肥计划
- 详细的每日饮食指南，包括餐厅就餐选择
- 渐进式的30天运动计划，适合在家进行
- 进度跟踪功能
- 响应式设计，适配各种设备

## 部署说明

### 方法一：直接访问

您可以直接访问已部署的网站：[30天减肥计划](https://your-username.github.io/weight-loss-plan/)

### 方法二：本地运行

1. 克隆此仓库：
   ```
   git clone https://github.com/your-username/weight-loss-plan.git
   ```

2. 打开`index.html`文件即可在浏览器中查看网站

### 方法三：部署到您自己的GitHub Pages

1. Fork此仓库
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 几分钟后，您的网站将在`https://your-username.github.io/weight-loss-plan/`上线

## 使用指南

1. 首先阅读"减肥科学"页面，了解减肥的基本原理
2. 查看"饮食计划"和"运动计划"，了解30天的详细安排
3. 使用"进度跟踪"页面记录您的减肥进度
4. 如有问题，请参考"常见问题"页面

## 自定义

如需根据个人情况调整计划：

1. 修改`diet-plan.html`和`exercise-plan.html`中的内容
2. 调整卡路里摄入量和运动强度
3. 保存更改并重新部署
